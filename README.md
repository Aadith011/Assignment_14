# Assignment_14

This is the backend API for a **Personal Website** built using **Node.js**, **Express**, and **MongoDB (Mongoose)**.  
It provides routes to:
- Fetch resume details for the frontend (Home & About pages)
- Save contact form submissions from the website

## Technologies Used
- **Node.js** – JavaScript runtime environment  
- **Express.js** – Web framework for building APIs  
- **MongoDB** – NoSQL database for storing data  
- **Mongoose** – Object Data Modeling (ODM) library for MongoDB  
- **CORS** – Middleware for handling cross-origin requests
